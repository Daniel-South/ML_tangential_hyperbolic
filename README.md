# ML_tangential_hyperbolic
Gain Intiution into the Output of the Tangential Hyperbolic Function tanh(x)  
  
Demonstrate the output of the tangential hyperbolic function over a range of input values.  
  
tanh(x) = (e^x - e^-x) / (e^x + e^-x)  
  
The output of tanh(x) is centered around zero and approaches -1 and 1 asymptotically.  
  
tanh(x) is used as an activation fuction in Recurrent Neural Networks, i.e. neural networks designed  
to process sequence data, such as stock prices or sequences of words, where order is significant.  
  
The zero-centered output of tanh(x) helps networks converge and learn faster.
